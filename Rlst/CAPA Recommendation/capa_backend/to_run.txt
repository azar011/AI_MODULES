Qdrant install Locally :-

1. https://github.com/qdrant/qdrant/releases?utm_source=chatgpt.com     --> [Download] - qdrant-x86_64-pc-windows-msvc.zip
2. Extract in C drive -->  C:\qdrant\
3. Open in cmd --> qdrant.exe
4. Qdrant starts on http://localhost:6333

On Ubuntu :-

1. https://github.com/qdrant/qdrant/releases?utm_source=chatgpt.com     --> [Download] - qdrant_1.18.2-1_amd64.deb
2. Open Terminal : sudo dpkg -i qdrant_1.18.2-1_amd64.deb
3. qdrant
4. if not : sudo mkdir -p /var/lib/qdrant/storage
